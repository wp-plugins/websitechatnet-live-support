== Installation ==

1. Download WebsiteChat-net.zip to your computer and add new plugin in WordPress administration.
2. Activate `WebsiteChat.net Live Support` through the 'Plugins' menu in WordPress.
3. [Get WebsiteChat.net account](http://websitechat.net/en/register)
4. You can review installation instruction at [WebsiteChat.net WordPress Live Support widget installation instructions](http://websitechat.net/en/3rd-party/wordpress)
